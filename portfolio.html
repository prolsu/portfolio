meApp

The "meApp" includes a location-based weather forecast, the New York Times top news, a random motivational quote, and a section to add personal notes. The user may choose to save any articles and quotes in a "favorites" section. APIs were mainly used for this project.

My Weather Dashboard

My Weather Dashboard displays current weather conditions along with a 5-day forecast. The user's searches will be stored locally and displayed upon running the app. A delete button is provided as well.

Password Generator

The Password Generator is a simple tool to create passwords on demand. User may choose to inquire a random 50-digit password comprised of numbers, letters, and special characters. A second option is provided which requires the user's criteria to build a password with up to 128 characters.
